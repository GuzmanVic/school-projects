Para insertar un texto centrado, utilizamos la etiqueta "center"
Para alinear un texto a la izquierda utilizamos la etiqueta "p ALIGN = left"
Para ainear un texto a la derecha utilizamos la etiqueta "p ALIGN = right"

Para alinear un texto justificado utilizamos la etiqueta "p ALIGN = justify" de esta manera, podemos visualizar el texto justificado, como por ejemplo, este mismo: Lorem ipsum dolor sit amet, consectetur adipisicing elit. Veritatis accusamus iste amet soluta in autem, nulla quia impedit eos voluptates neque deserunt nesciunt repellat consectetur inventore vero, quas aliquam aspernatur.
